# Supplementary-Material-for-D2GCDW
This repository provides the supplementary materials for the paper “Myocardial Infarction Detection with Incomplete Multi-View Data via Dual-Branch Gating Completion and Dirichlet Weighting (D2GCDW)”.
It includes the detailed proof of Proposition 2, additional experimental descriptions, and statistical validation results (paired t-tests and corresponding tables and figures), which are provided to support and complement the main manuscript.
