# Supplementary-Material-for-D2GCDW
This repository provides the supplementary materials for the paper “Myocardial Infarction Detection with Incomplete Multi-View Data via Dual-Branch Gating Completion and Dirichlet Weighting (D2GCDW)”.
The materials include the detailed proof of Proposition 2, additional experimental descriptions, and statistical validation results (paired t-tests with corresponding tables and figures), which serve to support and complement the main manuscript.
The complete supplementary content is compiled in the file Supplementary Material.pdf.
